🎓 AI & ML Interactive Tutor
An interactive web application built with Streamlit and Google Gemini to help beginners learn Machine Learning through conversation and live code examples.

🚀 Features
Conversational Learning: Ask any AI or ML question and get a simple, analogy-based explanation.

Live Code Snippets: Every explanation includes a clean Python code snippet using libraries like Scikit-Learn or Matplotlib.

Session Persistence: Keeps track of your chat history so you can build on previous questions.

Mobile-Friendly: Deployable via Streamlit Cloud for learning on any device.

🛠️ Tech Stack
Frontend: Streamlit

AI Brain: Google Gemini 1.5 Flash

Language: Python 3.9+

📋 Prerequisites
Before running the app, you will need:

A Google AI API Key. You can get one for free at Google AI Studio.

💻 Local Installation
If you want to run this on your own computer:

Clone the repository:

Bash
git clone https://github.com/YOUR_USERNAME/ai-ml-tutor.git
cd ai-ml-tutor
Install dependencies:

Bash
pip install -r requirements.txt
Run the app:

Bash
streamlit run app.py
🌐 Deployment
This app is optimized for Streamlit Community Cloud. To deploy your own version:

Push your code to a public GitHub repository.

Connect your GitHub account to Streamlit Cloud.

Select your repository and click Deploy.
